Pharma Night Prep CSV Pack

Main file:
- drugs.csv: website-ready drug revision data. One row = one drug within one class/problem context.

Suggested website fields:
- drug_name
- drug_class
- system
- problem_area
- mechanism
- exam_use
- side_effects
- exam_hint
- related_drugs
- related_classes
- revision_priority
- tags

Other files:
- drug_classes.csv: one row per class
- problems.csv: one row per clinical/exam problem group
- flashcards.csv: class-level flashcards
- quiz_items.csv: MCQ practice
- drugs_sample_first_100.csv: smaller test file

Safety note:
This content is for pharmacology exam revision only. It is not medical advice, and it contains no dosing or self-treatment guidance.
