Educational Disclaimer

Pharma Night Prep is made for educational revision only.

This app does not provide medical advice, diagnosis, treatment, prescription guidance, dosing guidance, or self-treatment instructions.

Students should always follow official textbooks, institutional guidelines, teachers, and qualified healthcare professionals.

Purpose of the App

Pharma Night Prep is a pharmacology revision tool for students. It helps users review drug classes, mechanisms, exam hints, flashcards, quizzes, and weak topics.

The app is designed for exam preparation and active recall practice.

Data Collection

Pharma Night Prep does not require login, account creation, email, phone number, or personal profile information.

The app does not intentionally collect personal information from users.

Local Storage

Some app features may save small study-related data inside the user's own browser using local storage.

  • Weak Review items
  • Saved wrong quiz answers
  • App cache for faster loading

This data stays on the user's device/browser. It is not sent to a separate database by this app.

Third-Party Hosting

The app is hosted using GitHub Pages. GitHub may process standard technical information required to deliver the website, such as basic request and browser information.

Advertising

Pharma Night Prep may display ads using Google AdMob. AdMob may collect device information, advertising identifiers, and usage data to show ads, measure ad performance, and prevent fraud.

Users can learn more about Google's advertising practices and manage ad personalization through their Google account settings.

Content Accuracy

The content is intended for quick revision and may be simplified for exam preparation. Users should verify important academic or clinical information from official sources.

Clearing Saved Data

Users can clear saved Weak Review items using the Clear List button inside the app.

Users may also clear browser site data/cache from their browser settings.

Contact

For project updates or issues, users can visit the GitHub repository where the app is hosted.
