# Pharma Night Prep

**Pharma Night Prep** is a smart pharmacology revision web app designed for quick exam preparation, last-night revision, and high-yield drug review.

It helps students revise pharmacology using searchable drug cards, system-wise indexing, flashcards, quiz mode, exam score summaries, and weak-topic review.

## Live App

[Open Pharma Night Prep](https://mimruwa.github.io/Pharmacology_Exam_Prep_night/)

## Features

* Search drugs, drug classes, systems, mechanisms, and exam hints
* Filter drugs by body system
* Browse pharmacology topics through a revision index
* View high-yield drug cards
* Open detailed drug information
* Flashcard mode for quick revision
* Quiz mode with multiple-choice questions
* 10-question exam mode with final score summary
* Weak Review list for wrong quiz answers
* Update App button to refresh the latest version
* Installable as a Progressive Web App
* Works using a simple CSV-based drug database

## How It Works

The app loads pharmacology revision data from `drugs.csv`.

Each drug entry can include:

* Drug name
* Drug class
* System
* Mechanism
* Exam use
* Side effects
* Exam hint
* Related drugs

The app then automatically creates searchable cards, flashcards, quizzes, and weak review items from that data.

## App Sections

### Dashboard

Shows the total number of systems, drug classes, and high-yield drugs.

### Revision Index

Allows students to browse drugs by system, class, or A-Z order.

### High-Yield Drug Cards

Displays compact exam-focused drug cards for quick revision.

### Flashcards

Helps students actively recall drug mechanisms, uses, side effects, and exam hints.

### Quiz Mode

Generates multiple-choice questions from the existing drug database.

### Exam Mode

Allows students to complete a fixed question set and view a final score summary.

### Weak Review

Automatically saves wrong quiz answers so students can revise weak topics later.
